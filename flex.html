FlexBox

O que é o FlexBox?

Assim como diversas propriedades da estilizaçao em cascata (CSS), o FlexBox é um módulo que contém propriedades, algumas sendo declaradas como elementos-pai "flex-container" e os elementos-filhos "flex-itens"

O flex-container, como o próprio nome sugere, se trata dos limites da caixa criada, como por exemplo a forma de "fora". Dessa maneira, os itens que serão inseridos como conteúdo podemm ser organizados de sua preferência. Veja os exemplos a seguir.

O que é o Grid

É um sistema de estruturação de layout que permite organizar conteúdo em linhas e colunas, formando uma grade, trazendo também maior organização e responsividade para o projeto em questão. Ele também funciona com a propriedade container, e para comerçarmos a urilizalo, devemos definí-lo como senod do tipo grid.
<.container{ display: grid; }>

Por padrão ele criará uma grid do tipo block, ocupando uma linha inteira, porém pode-se usar o "display: inline-grid;" para utilizar containers.

Como se trata de grades, é necesário informar o número de linhas e colunas desejadas. Para isso utlizamos grid-template-rows e para linhas usamos o grid-template-colums respectivamente.
